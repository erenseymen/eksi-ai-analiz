Gemini API Key Nasıl Alınır?

  1. Google AI Studio sayfasına gidin.
  2. Google hesabınızla giriş yapın.
  3. "Create API key" butonuna tıklayın.
  4. Oluşturulan anahtarı kopyalayıp yukarıdaki kutucuğa yapıştırın ve Kaydet'e basın.

Not: API anahtarınız sadece tarayıcınızın yerel hafızasında saklanır ve sadece Google sunucularına istek atmak için kullanılır.
